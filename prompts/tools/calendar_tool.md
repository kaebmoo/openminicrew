จัดการ Google Calendar (ดู/เพิ่ม/ลบนัดหมาย). ใช้เมื่อ user ถามตารางงาน นัดหมาย หรือให้ลงปฏิทิน. ไม่ใช่สำหรับการจดงาน (ใช้ todo) หรือตั้งเตือนปลุก (ใช้ reminder/schedule). เช่น 'พรุ่งนี้มีนัดไหม', 'ดึงตารางงาน', 'เพิ่มนัดประชุม 10 โมง'.

Action:
- list: ดูนัดหมายที่กำลังจะมาถึง
- add: เพิ่มนัดหมาย ต้องมี date (YYYY-MM-DD ปี ค.ศ.), start_time (HH:MM 24h), title. end_time ถ้าไม่ระบุจะ default + 1 ชม.
- delete: ใช้ event_ref (ลำดับจาก list หรือ event_id)

แนวทางการดึงข้อมูลจากข้อความผู้ใช้สำหรับ action=add:
- title: ชื่อ/หัวข้อสั้น ๆ — ห้ามใส่ URL หรือสถานที่ลงในนี้
- description: URL (Zoom/Meet/Teams), agenda, รายละเอียด, หมายเหตุ — Google Calendar จะแปลง URL เป็นปุ่ม join ให้
- location: สถานที่จริงเท่านั้น (ตึก/ห้อง/ที่อยู่). ถ้าเป็นประชุมออนไลน์ปล่อยว่าง

IMPORTANT: แปลงวันที่ไทย/พ.ศ. → YYYY-MM-DD (ค.ศ.) ก่อนเรียก เช่น '18 พฤษภาคม 2569' → '2026-05-18'.
