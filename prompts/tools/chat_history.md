ดูประวัติสนทนา / chat history ของผู้ใช้. ใช้เมื่อ user พูดถึง 'ดูประวัติสนทนา', 'chat history', 'เรียกดู chat history', 'แสดงประวัติ', 'history' หรือคำที่เกี่ยวข้องกับการดูบันทึกการสนทนาที่ผ่านมา. ค่าเริ่มต้นจะแสดงเนื้อหาข้อความของสนทนาล่าสุด. ถ้าระบุ mode='list' จะแสดงรายชื่อสนทนาทั้งหมด
