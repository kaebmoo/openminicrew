Privacy Policy

OpenMiniCrew - AI-Powered Telegram Bot

Last updated: March 26, 2026

Overview

OpenMiniCrew is an open-source Telegram bot that helps users manage everyday tasks through AI-powered tools. This policy explains what data the application collects, how it is used, and how you can control or delete your data.

Data We Collect

OpenMiniCrew collects and processes the following data only when you use the corresponding features:

  • Telegram profile: Your Telegram chat ID and display name, used to identify your account and deliver messages.
  • Messages: Text messages you send to the bot are processed to determine which tool to use. Chat history may be stored temporarily to provide conversational context, subject to your consent.
  • Location: If you share your location, it is stored temporarily to provide location-based features (nearby places, traffic). Location data expires automatically based on a configurable time limit and can be deleted at any time.
  • Phone number and national ID: Stored only if you voluntarily provide them for PromptPay QR generation. These are encrypted at rest.
  • Expense records: Amounts, categories, and notes you record through the expense tracker. Notes are encrypted at rest.
  • Google account data: See the dedicated section below.

Google User Data

OpenMiniCrew accesses Google APIs (Gmail API and Google Calendar API) only after you explicitly authorize access through the standard Google OAuth 2.0 flow.

The application accesses:

  • Gmail (read-only): Message headers and content are read to generate email summaries. The application does not send, modify, or delete your emails.
  • Google Calendar: Calendar events are read to show your schedule. Events can be created or deleted only when you explicitly request it.

How Google data is used: Email content and calendar data are processed in memory to generate summaries and are not stored permanently. Only minimal metadata (message IDs for deduplication) is retained temporarily.

Sharing: Your Google data is never shared with, sold to, or disclosed to any third party. It is processed only by the LLM provider (Google Gemini or Anthropic Claude) as configured by the bot operator, solely for the purpose of generating responses to your requests.

OpenMiniCrew's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

Data Security

OpenMiniCrew implements the following security measures:

  • Sensitive identifiers (phone numbers, national IDs) are encrypted at rest using field-level encryption.
  • Google OAuth tokens are encrypted at rest in per-user credential files.
  • Per-user API keys are encrypted before storage.
  • Operational logs use minimized metadata (hashes and sizes) instead of raw content.
  • The Telegram webhook endpoint is protected by a secret token verified on every request.

Data Retention

Data is not kept indefinitely. The bot operator configures retention periods for chat history, operational logs, and other temporary data. Automated cleanup jobs remove expired data on a daily schedule. Location data has an independent time-to-live setting and expires automatically.

Your Rights and Controls

You have full control over your data through the following commands:

  • /privacy — View a summary of your consent state and data status.
  • /consent gmail off — Revoke Gmail access consent.
  • /consent location off — Revoke location consent and delete saved location.
  • /consent chat off — Stop chat history storage and delete existing history.
  • /disconnectgmail — Remove Gmail authorization and delete the stored token.
  • /clearlocation — Delete your saved location immediately.
  • /delete_my_data confirm — Permanently delete your user-linked operational data while intentionally retaining a minimal governance audit trail.

You can also revoke OpenMiniCrew's access to your Google account at any time through your Google Account permissions page.

Third-Party Services

OpenMiniCrew integrates with the following third-party services to provide its features:

  • Telegram Bot API (message delivery)
  • Google APIs: Gmail and Calendar (email and calendar features)
  • Google Gemini or Anthropic Claude (AI language processing)
  • Google Maps Platform (places and traffic)
  • Tavily or DuckDuckGo (web search)
  • Bank of Thailand API (exchange rates)

Each service is governed by its own privacy policy. Data sent to these services is limited to what is necessary for the specific feature.

Open Source

OpenMiniCrew is open-source software. The full source code is available at github.com/kaebmoo/openminicrew. You can inspect exactly how data is handled.
