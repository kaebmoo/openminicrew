เริ่มต้นใช้งาน

ลงทะเบียน

พิมพ์ /start เพื่อลงทะเบียนและเริ่มใช้งาน Bot จะแสดงนโยบายความเป็นส่วนตัวแบบย่อ พร้อมให้เลือกอนุญาตหรือไม่อนุญาตการเก็บประวัติสนทนาผ่านปุ่มกด (Inline Keyboard) เปลี่ยนใจได้ทุกเมื่อผ่าน /consent

ตั้งค่าข้อมูลพื้นฐาน (แนะนำ)

  • /setname <ชื่อ> — ตั้งชื่อที่แสดง
  • /setphone <เบอร์โทร> — บันทึกเบอร์โทรสำหรับ PromptPay QR
  • /setid <เลขบัตร 13 หลัก> — บันทึกเลขบัตรประชาชนสำหรับ PromptPay QR

เชื่อมต่อ Gmail (สำหรับฟีเจอร์อีเมลและปฏิทิน)

พิมพ์ /authgmail แล้วกดลิงก์เพื่อ authorize ผ่าน Google OAuth

วิธีสั่งงาน

OpenMiniCrew รองรับ 2 วิธีในการสั่งงาน:

1. พิมพ์คำสั่งตรง — ขึ้นต้นด้วย / เช่น /email, /fx USD ไม่เสีย LLM token

2. พิมพ์อิสระ — พิมพ์ภาษาธรรมชาติ AI จะเลือก tool ที่เหมาะสมให้อัตโนมัติ เช่น "สรุปเมลให้หน่อย", "เงินดอลลาร์วันนี้เท่าไหร่", "รถติดไหม สยามไปสีลม", "ดู chat history", "เปิดยินยอมบันทึกประวัติสนทนา"

3. ส่งรูปถ่าย — ถ่ายรูปบิล ใบเสร็จ หรือ slip ส่งมา Bot จะอ่านและบันทึกรายจ่ายอัตโนมัติ

อีเมล

Gmail ส่วนตัว

คำสั่ง: /gmail หรือ /email

ต้องเชื่อมต่อ Gmail ก่อนด้วย /authgmail

คำสั่ง ความหมาย
/email สรุปอีเมลที่ยังไม่ได้อ่านวันนี้
/email 3d สรุปอีเมลย้อนหลัง 3 วัน
/email 7d สรุปอีเมลย้อนหลัง 7 วัน
/email 30d สรุปอีเมลย้อนหลัง 30 วัน
/email force สรุปใหม่ทั้งหมด (รวมที่เคยสรุปแล้ว)
/email บัตรเครดิต ค้นหาอีเมลเกี่ยวกับบัตรเครดิต
/email from:ktc.co.th 7d ค้นหาอีเมลจาก KTC ย้อนหลัง 7 วัน
/email force from:grab.com 30d สรุปใหม่เฉพาะอีเมลจาก Grab 30 วัน

สามารถใช้ร่วมกันได้ เช่น ค้นหา + ช่วงเวลา + force

อีเมลที่ทำงาน (IMAP)

คำสั่ง: /wm หรือ /workmail

ต้องตั้งค่า IMAP ก่อน: - /setkey work_imap_host <host> — เช่น mail.company.com - /setkey work_imap_user <email> — เช่น yourname@company.com - /setkey work_imap_password <password>

คำสั่ง ความหมาย
/wm สรุปอีเมลงานวันนี้
/wm 7d ย้อนหลัง 7 วัน
/wm from:boss@company.com เฉพาะจากหัวหน้า
/wm subject:ประชุม เฉพาะเรื่องประชุม
/wm force 3d สรุปใหม่ย้อนหลัง 3 วัน
/wm folder:Sent ดูโฟลเดอร์อื่น

รองรับไฟล์แนบ: PDF, DOCX, XLSX, CSV, TXT (สูงสุด 5 MB ต่อไฟล์, อ่านเนื้อหาสูงสุด 3 ไฟล์แนบต่ออีเมล)

ตัวอย่าง: ตั้งค่าอีเมลที่ทำงาน (IMAP) สำหรับ NT

สำหรับผู้ใช้อีเมล NT (National Telecom) @ntplc.co.th ให้ตั้งค่าทีละบรรทัดดังนี้:

/setkey work_imap_host mail.ntplc.co.th
/setkey work_imap_user yourname@ntplc.co.th
/setkey work_imap_password รหัสผ่านของคุณ

เปลี่ยน yourname เป็นชื่อบัญชีอีเมลจริงของคุณ และใส่รหัสผ่านจริง (ข้อความจะถูกลบจากแชทอัตโนมัติ)

เมื่อตั้งค่าครบทั้ง 3 รายการแล้ว ใช้ /wm เพื่อสรุปอีเมลงานได้ทันที

สำหรับองค์กรอื่นที่ใช้ IMAP ให้เปลี่ยน host, user, password ตามข้อมูลของ mail server ขององค์กร (port 993 SSL เป็นค่าเริ่มต้น)

Smart Inbox — หา Action Items จากอีเมล

คำสั่ง: /inbox

คำสั่ง ความหมาย
/inbox วิเคราะห์อีเมลล่าสุดหา action items
/inbox mode auto เปลี่ยนเป็น auto — สร้าง todo อัตโนมัติ
/inbox mode confirm เปลี่ยนเป็น confirm (ค่าเริ่มต้น) — แสดงผลแต่ไม่สร้าง todo

การเงิน

บันทึกรายจ่าย

คำสั่ง: /exp หรือ /expense

คำสั่ง ความหมาย
/expense 120 อาหาร ก๋วยเตี๋ยว บันทึก 120 บาท หมวดอาหาร หมายเหตุ ก๋วยเตี๋ยว
/expense 65 เครื่องดื่ม เบียร์ บันทึก 65 บาท หมวดเครื่องดื่ม
/expense 350 บันเทิง บันทึก 350 บาท หมวดบันเทิง
/expense list ดูรายจ่ายล่าสุด (แสดงเป็นลำดับ 1, 2, 3)
/expense delete 3 ลบรายจ่ายลำดับที่ 3
/expense edit 3 120 แก้ราคารายจ่ายลำดับที่ 3 เป็น 120 บาท
/expense summary month สรุปรายจ่ายเดือนนี้
/expense summary today สรุปรายจ่ายวันนี้
/expense summary 7d สรุปรายจ่าย 7 วันล่าสุด
/expense summary month เครื่องดื่ม สรุปเฉพาะหมวดเครื่องดื่ม
/expense summary month กาแฟ ค้นหาคำว่า "กาแฟ" ในหมายเหตุ

หมวดหมู่ที่ใช้ได้: อาหาร, เครื่องดื่ม, เดินทาง, ช็อปปิ้ง, ของใช้, สาธารณูปโภค, สุขภาพ, บันเทิง, การศึกษา, โอนเงิน, ทั่วไป

บันทึกจากรูปถ่าย: ถ่ายรูปบิล ใบเสร็จ หรือ slip ส่งมาเลย Bot จะใช้ Gemini Vision วิเคราะห์และบันทึกให้อัตโนมัติ ถ้ามีหลายรายการในใบเสร็จ Bot จะถามว่าจะแยกรายการหรือรวมเป็นรายการเดียว สามารถใส่ caption กำกับรูปได้ เช่น "กินข้าวกับเพื่อน"

Service charge / VAT: ถ้าใบเสร็จมี service charge หรือ VAT ระบบจะเฉลี่ยเข้ารายการอัตโนมัติตามสัดส่วน (ไม่แยกบรรทัด SC/VAT ออกมา)

แก้ไข/ลบรายจ่าย: ใช้ /expense list เพื่อดูลำดับ แล้ว /expense delete <ลำดับ> เพื่อลบ หรือ /expense edit <ลำดับ> <ราคาใหม่> เพื่อแก้ราคา

สถานที่และการเดินทาง

ค้นหาสถานที่

คำสั่ง: /places หรือ /nearby

คำสั่ง ความหมาย
/places ร้านกาแฟแถวสยาม ค้นหาร้านกาแฟแถวสยาม
/places ร้านอาหารแถวนี้ ค้นหาใกล้ตำแหน่ง GPS ของคุณ
/places โรงพยาบาลใกล้ลาดพร้าว ค้นหาโรงพยาบาล
/places ร้านอาหารเปิดอยู่ เฉพาะร้านที่เปิด
/places กาแฟ 4.5 ดาวขึ้นไป กรองตาม rating
"แถวนี้คือที่ไหน" บอกตำแหน่งปัจจุบันจาก GPS (reverse geocode)

ส่งตำแหน่ง GPS มาก่อนเพื่อค้นหาใกล้ตัว: กดปุ่ม 📎 (แนบไฟล์) > เลือก Location ผลลัพธ์แสดง rating รีวิว เวลาเปิด-ปิด เบอร์โทร และลิงก์ Google Maps

เช็คเส้นทางและจราจร

คำสั่ง: /traffic หรือ /route

คำสั่ง ความหมาย
/traffic สยาม ไป สีลม เช็คเส้นทาง สยาม → สีลม
/traffic MBK to Asiatique เช็คเส้นทางภาษาอังกฤษ
/traffic สยาม → อโศก ใช้ลูกศรแทน "ไป" ได้
/traffic ไปสนามบิน ต้นทาง = ตำแหน่ง GPS ปัจจุบัน
/traffic สยาม ไป สีลม ไม่ขึ้นทางด่วน หลีกเลี่ยงทางด่วนทั้งหมด
/traffic จันทน์ ไป แจ้งวัฒนะ ไม่ขึ้น ดอนเมืองโทลเวย์ หลีกเลี่ยงเฉพาะทางด่วนที่ระบุ

โหมดการเดินทาง: Bot เลือกอัตโนมัติจากคำที่พิมพ์ หรือใช้คำต่อท้าย: - ขับรถ (ค่าเริ่มต้น) - มอไซค์ / มอเตอร์ไซค์ — มอเตอร์ไซค์ - เดิน — เดินเท้า - รถเมล์ / BTS / MRT / รถไฟฟ้า — ขนส่งสาธารณะ

แสดงระยะทาง เวลาปกติ เวลาจริงรวม traffic เส้นทางโดยย่อ และลิงก์เปิดใน Google Maps

ตั้งเวลาอัตโนมัติ

คำสั่ง: /schedule

ตั้งให้ Bot รัน tool อัตโนมัติตามเวลาที่กำหนด

คำสั่ง ความหมาย
/schedule ดูรายการ schedule ทั้งหมด
/schedule add 07:00 gmail_summary ทุกวัน 7 โมง สรุปอีเมล
/schedule add weekday 08:30 traffic สุขุมวิท จ-ศ 8:30 เช็คจราจร
/schedule add weekly mon 09:00 news_summary ทุกจันทร์ 9 โมง สรุปข่าว
/schedule add monthly 1 07:00 exchange_rate ทุกวันที่ 1 เช็คอัตราแลกเปลี่ยน
/schedule add once 2026-03-30 09:00 gmail_summary ครั้งเดียว
/schedule remove 3 ลบ schedule ID 3

ความถี่ที่ใช้ได้: daily (ทุกวัน), weekday (จ-ศ), weekly (ทุกสัปดาห์ ระบุวัน), monthly (ทุกเดือน ระบุวันที่), once (ครั้งเดียว ระบุวันที่)

Tool ที่ schedule ได้: gmail_summary, work_email, news_summary, exchange_rate, traffic, lotto, places และ tool อื่นทุกตัว

API Keys

คำสั่ง ความหมาย

ข้อความที่มี API key จะถูกลบจากแชทอัตโนมัติเพื่อความปลอดภัย

Service ที่รองรับ: gemini, anthropic (Claude), tavily, tmd, google_maps, work_imap_host, work_imap_user, work_imap_password และอื่นๆ

ตัวอย่าง: ตั้งค่า API Key สำหรับ LLM

Gemini (Google AI Studio): 1. ไปที่ https://aistudio.google.com/apikey 2. กด "Create API Key" 3. copy key ที่ได้ แล้วพิมพ์ใน bot:

/setkey gemini AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

Claude (Anthropic): 1. ไปที่ https://console.anthropic.com/settings/keys 2. กด "Create Key" 3. copy key ที่ได้ แล้วพิมพ์ใน bot:

/setkey anthropic sk-ant-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

หลังตั้งค่าแล้ว เปลี่ยน LLM ที่ใช้งานด้วย /model gemini หรือ /model claude

พิมพ์อิสระ: สามารถจัดการ consent ด้วยภาษาธรรมชาติได้ เช่น "เปิดยินยอมบันทึกการสนทนา", "ปิดการบันทึกตำแหน่ง", "ดูสถานะ consent"

On-demand consent: เมื่อใช้ฟีเจอร์ที่ต้องการ consent (เช่น ค้นหาสถานที่ใกล้ตัว) แต่ยังไม่เคยให้ consent ระบบจะถามอนุญาตผ่านปุ่มกดก่อนดำเนินการ
