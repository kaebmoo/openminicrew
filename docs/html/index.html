OpenMiniCrew

AI-Powered Personal Assistant Telegram Bot

About

OpenMiniCrew is an open-source Telegram bot framework that uses LLM-powered routing to connect users with modular tools for everyday tasks. It supports multiple AI providers including Google Gemini and Anthropic Claude.

Features

  • Gmail and Google Calendar integration for email summaries and scheduling
  • Expense tracking with receipt OCR via Gemini Vision
  • PromptPay QR code generation
  • Real-time traffic and places search via Google Maps
  • Web search and news summaries
  • Reminders, to-do lists, and scheduled automation
  • Thai unit conversion (land area, gold weight)
  • Exchange rate lookup

Google API Usage

OpenMiniCrew uses Google APIs (Gmail API and Google Calendar API) to provide email summary and calendar management features. Access to your Google data requires explicit authorization through OAuth 2.0 and can be revoked at any time using the /disconnectgmail command or through your Google Account permissions.

OpenMiniCrew's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.
